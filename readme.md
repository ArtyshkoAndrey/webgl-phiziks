# Программа для работы с атомамами на WebGL

Реализованно на Vue.JS, Electron.JS, Three.JS

## Установка

Использутей пакетный менеджер [yarn](https://yarnpkg.com/lang/en/) для установки.

```bash
yarn
```

## Испрользование

```bash
yarn run dev
```
